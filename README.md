# HTML to PDF Converter (Multilingual)

Convert HTML files to PDF with full **multilingual support** including **English, Hindi, Tamil, Chinese, and Arabic** using Python and WeasyPrint.

## Features

- Convert multiple HTML files from an **input folder** to PDF in an **output folder**.
- Supports **Unicode fonts** using Noto and DejaVu fonts.
- Runs in a **Docker container** for easy deployment on Windows, Linux, and OCP.
- Automatically logs conversion status for each file.
